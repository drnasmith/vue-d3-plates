Vue.js, d3.js and tailwind CSS Sample Group selection demo

Description

This demo combines a couple of things.

Use of SVG to draw plates, edit the plate geometry and click "Update Plate Geometry" to see changes.

Drop width and height are scaling factors 0-1 to accurately reflect different plate types).

Clicking on each cell shows how we could add sample to a SampleGroup. Selected sample ids shown above the plate view.

Changing the colour scale demonstrates the use of different filters (viridis, rgb, threshold).

Threshold is shown as a percentage and filters "scores" from 0.0 - 1.0 (100%).
